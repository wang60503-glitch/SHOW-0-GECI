# 字体颜色和图标颜色说明

整理时间：2026-05-18 16:51

这份只记录当前桌面歌词界面里和歌词框相关的字体颜色、歌词颜色、图标颜色。

## 1. 没唱歌词颜色

没唱歌词是深蓝色系，上下有轻微渐变：

- 顶部 `0%`：`#076ca4`
- 到 `25%`：`#0871aa`
- 到 `75%`：`#0871aa`
- 底部 `100%`：`#0978b2`

效果意思：

- 上面 `25%` 稍微深一点。
- 中间保持原深蓝。
- 底部 `25%` 稍微浅一点。

## 2. 唱完歌词颜色

唱完歌词是浅蓝、白色、浅蓝的竖向渐变：

- `0%`：`#4ffbff`
- `25%`：`#4ffbff`
- `50%`：`#ffffff`
- `100%`：`#00ffff`

效果意思：

- 上面 `0%-25%` 是很浅的蓝。
- 中间过渡到白色。
- 底部 `50%-100%` 是浅蓝。

唱完歌词还有一点提亮：

- `brightness(1.14)`
- `saturate(1.08)`

唱完歌词每个字的笔画外有很细的黑色边：

- 黑边：`rgba(0, 0, 0, 0.98)`
- 黑边粗细：`0.55 px`

这个黑边不是给整个字加方框，是给每个字的笔画加细边，用来做阴影和立体感。

## 3. 没唱和唱完的共同文字效果

两种歌词都有基础细黑边和阴影：

- 基础黑边：`0.5 px rgba(0, 0, 0, 0.96)`
- 阴影 1：`0 1px 0 rgba(190, 255, 255, 0.14)`
- 阴影 2：`0 2px 4px rgba(0, 0, 0, 0.22)`

## 4. 图标默认颜色

图标默认颜色：

- `#d8dddd`

意思：

- 默认是灰白色。
- 不要发光。
- 不要默认蓝色。

## 5. 鼠标悬停图标颜色

鼠标放到图标上时：

- 最终悬停颜色：`#f7ffff`

说明：

- 代码里前面也有一个 `#ffffff` 悬停色。
- 但后面的悬停规则覆盖了它。
- 所以当前实际生效的是 `#f7ffff`。

## 6. 蓝色激活图标颜色

部分图标打开后会变蓝：

- 蓝色：`#1aa9ff`

当前用这个蓝色的图标：

- 五角星打开/激活时
- 鱼图标打开/激活时

## 7. 心形图标颜色

心形点击动画时：

- 动画颜色：`#ff4f2e`

心形处于喜欢状态时：

- 喜欢状态颜色：`#ff7d9e`

## 8. 页面基础文字色

桌面歌词页面基础文字色：

- `#0aa8db`

说明：

- 这是页面根层的基础色。
- 歌词本身有自己的颜色规则，主要看上面的没唱/唱完颜色。
