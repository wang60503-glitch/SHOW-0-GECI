# 歌词框相关代码位置

这份只记录歌词框相关代码在哪里，不包含完整图标代码。

## 主文件

`G:\SHOW-0\show0-new-ui\desktop-lyrics-project\desktop-lyrics-app.tsx`

## 关键常量

- `FRAME_WIDTH = 506`
  - 可见歌词框宽度。

- `FRAME_HEIGHT = 327`
  - 可见歌词框最大高度。

- `LYRIC_DISPLAY_TOP = 35`
  - 隐藏歌词显示框顶边位置。

- `LYRIC_DISPLAY_BOTTOM_Y = 307`
  - 最大状态下隐藏歌词显示框底边位置。

- `LYRIC_MAX_FONT_SIZE = 96`
  - 歌词最大字号。

- `LYRIC_MIN_FONT_SIZE = 22`
  - 歌词最小字号。

- `FIRST_LYRIC_ROW_LEFT = 14`
  - 第一排歌词左侧安全距离。

- `SECOND_LYRIC_ROW_LEFT = FIRST_LYRIC_ROW_LEFT`
  - 第二排歌词左侧安全距离。

## 关键函数

- `getLyricMetrics(fontSize)`
  - 根据字体大小计算歌词框高度、行距、底部距离。

- `getLyricFontSizeForWindowHeight(windowHeight)`
  - 根据真实窗口高度反算歌词字体大小。

- `startLyricFontResize(edge, event)`
  - 处理上下边缘拖动改变歌词字号。

## 关键结构

- `desktopLyricsArea`
  - 隐藏歌词显示框区域。
  - 当前顶边为 `35 px`。
  - 当前底边跟随窗口高度变化。

- `desktopLyricsRows`
  - 两行歌词布局容器。
  - 使用三段结构：
    - 第一段放第一排歌词
    - 中间段自动撑开
    - 第三段放第二排歌词

- `desktopLyricsRow:first-child`
  - 第一排歌词。
  - 固定靠左上。

- `desktopLyricsRow:last-child`
  - 第二排歌词。
  - 固定靠右下。

- `desktopLyricsText`
  - 每行自己的文字字框。
  - 只包文字，不是整行大框。
